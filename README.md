# 🚗 CarChrono

CarChrono is a modern, responsive web application designed to showcase car details in an alphabetical A–Z format. It provides a smooth browsing experience for users and a secure admin dashboard for managing car data.

---

## ✨ Features

* 🔤 Browse cars alphabetically from A to Z
* 🖼️ View car images, names, brands, and descriptions
* 🔐 Secure admin login
* ➕ Add, ✏️ edit, and ❌ delete car details
* 📊 Admin dashboard with car statistics
* 🎨 Clean UI with animations
* 📱 Fully responsive design

---

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router), HTML, CSS, JavaScript
* **Backend:** Supabase (Database & Authentication)
* **Hosting:** Vercel
* **Code Management:** GitHub

---

## 🗄️ Database Schema

**Table:** `cars`

* `id` (uuid)
* `car_name` (text)
* `brand` (text)
* `image_url` (text)
* `description` (text)
* `created_at` (timestamp)

---

## 🚀 Deployment

The application is deployed on **Vercel** and connected securely to **Supabase** using environment variables.
