# **App Name**: CarChrono

## Core Features:

- Car Listing: Display cars grouped alphabetically, showing image, name, brand, and description.
- Smooth Scroll Navigation: Implement A-Z smooth scroll for quick navigation through the car list.
- Admin Authentication: Secure the admin page using Supabase email/password authentication.
- CRUD Operations: Enable adding, editing, and deleting cars via the admin dashboard, using Supabase for data persistence.
- Dashboard Statistics: Show the total car count in the dashboard.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and sophistication.
- Background color: Very light blue (#E8EAF6), almost white, to provide a clean backdrop.
- Accent color: Amber (#FFC107) for interactive elements.
- Body and headline font: 'Inter', a sans-serif font for a modern, neutral look.
- Use clean, outlined icons for navigation and actions.
- Responsive grid layout for car cards and dashboard elements.
- Use subtle fade-in effects for car cards and hover scale for buttons.
- Dark theme primary color: Dark gray (#303030).
- Dark theme background color: Black (#000000).
- Dark theme text color: Light gray (#FAFAFA).
